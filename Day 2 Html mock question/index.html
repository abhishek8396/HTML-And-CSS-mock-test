Question- Difference between HTML tag and element?

Answer- HTML Tags: Tags are the starting and ending parts of an HTML element. They begin with < symbol and end with > symbol. Whatever written inside < and > are called tags.
HTML elements: Elements enclose the contents in between the tags. They consist of some kind of structure or expression. It generally consists of a start tag, content and an end tag.

Question- Explain br tag?

Answer- Br tag: The Line Break element. The Br tag HTML element produces a line break in text (carriage-return). It is useful for writing a poem or an address, where the division of lines is significant.

Question- Explain the different types of list we have in html?

Answer- There are three list types in HTML: unordered list — used to group a set of related items in no particular order. ordered list — used to group a set of related items in a specific order. description list — used to display name/value pairs such as terms and definitions.

Question- What are Semantic Elements?

Answer-A semantic element clearly describes its meaning to both the browser and the developer.
Examples of non-semantic elements: div tag and span tag - Tells nothing about its content.
Examples of semantic elements: form, table, and article - Clearly defines its content.
